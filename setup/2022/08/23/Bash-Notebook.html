## What To Do Next?

Great!  You have setup your repo.  Now its time to start writing content.  Some helpful links:

- [Writing Blogs With Jupyter](https://github.com/fastai/fastpages#writing-blog-posts-with-jupyter)

Hacks

  1. Is there anything we use to verify tools we install? Think about versions.
  • To verify the tools we install, we can use --version commands (for example python --version)
  1. Is there anything we could verify with Anaconda?
  • You can check the version of Anaconda using conda -version. You can also type conda list to obtain the packages installed using anaconda.
  1. How would you update a repository?
  • To update a repository in VSCode, you can click the "Source Control" tab on the left and push changes. Remember to include a message (it is just a note to yourself on what you changed, but GitHub won't push it until there is that message) when you are pushing changes.
  1. Really cool would be automating a procedure from installation.
  • You can create scripts using coding languages (such as Bash and Python, two languages that we have been introduced to) to automate installation procedures, so you don't have to do any extra work!!
